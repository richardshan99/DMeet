DMeet直面是一个直接面对面交友的社交平台。我们建立了一套激励约束机制，保障用户见面过程的真实、效率和安全性。区别于其它交友app，DMeet是”先见面，后聊天“，看过资料后直接发起见面邀约，我们相信一切真实的了解从见面开始。。

我们从以下几个方面来保障见面的真实可靠性：
1、所有用户见面前都要求提供真实的头像和身份认证，学历认证
2、我们在不同城市提供多个见面场所供选择，一般是咖啡店，餐厅等。您可以选择就近的见面地点发起邀约，双方可以有一次更改机会，确认后则不能更改，只能取消。
3、见面双方需支付一定的保证金，在约定时间地点签到后即可退还。如一方未履约签到，保证金归签到履约方作为补偿。
4、邀约发发起方可送上见面红包作为对方路费补贴和付出时间的感谢。
